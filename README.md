# zkdjs (中考倒计时)
这是一个诞生于作者初二课堂的倒计时程序，陪伴了我们全班同学走过了整整一年，最终在我自己参加中考那天完成了使命。

## 📖 背景故事
- **起缘**：初二那年，看着学长学姐冲刺中考，我突然想做一个倒计时牌放在班级电脑上。于是用 VB.NET 写出了第一版。
- **陪伴**：之后的一年里，我们把它添加进班级电脑的开机启动项，看着天数一天天减少。它成了我们班共同的记忆坐标。
- **终章**：2025年6月28日，中考第一天。当倒计时显示 **“0天”** 时，我们知道，它完成了自己的使命。

## 📂 项目状态
**此仓库已于2026年3月被标记为归档**。它已不再维护，但代码将永远保留这段青涩和回忆。 

## ✨ 功能
- 在屏幕左上角显示距离中考的剩余天数。

## 🛠️ 技术栈
VB.NET
